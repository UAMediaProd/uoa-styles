New Cues & Directions

Directions — UI

This is a Vanilla Direction Box

The vanilla adx-direction can be used for directions that don't fall into the following categories. Directions are to be used not as a purely visual element - but to indicate to students they should leave the page to complete the directions.

This is an Information Direction

Use the adx-direction-info to direct students to additional information. This would include external websites or documents.

This is a Question Box

Use the adx-direction-question to direct students to answer questions or supply additional information. This could include forms for services, sign-up information etc.

This is a Warning Box

Use the adx-direction-warning to provide students a warning to engage with something external. This would be useful for anything that is timeline or deadline driven

This is a Positive Box

Use the adx-direction-positive to provide students with postive information or messages. This would include external websites or documents.

This is a Negative Box

Use the adx-direction-info to provide students with direction NOT to do something. This would include external websites or documents.

Directions — Learning

Use the following direction boxes to markup specific directions for students. These are callouts to complete specific tasks external to the page and flow of the lesson.

This is a Assessment Direction

Use the adx-direction-assessment class to define information on the page about assessment tasks. This would specifically include directions to complete the task itself, or elements of the task.

This is a Discussion Direction

Use the adx-direction-discussion class to alert students to tasks that relate to discussions. This maybe located on another page - but if context information is suitable, i.e. the information in this lesson is the logical point to carry out and contribute to the discussion, then use this direction.

This is a Extra Direction

Use the adx-direction-extra class to alert students to extra tasks. These are not required but could be seen as extensions beyond the content and activities required in this course. This may be useful for "parking" concepts and ideas that students aren't required to complete but may be useful in the longer term.

This is a Interactive Direction

Use the adx-direction-extra class to alert students to interactive activities and ways to engage with other students. Can be used for synchronous sessions on Zoom or on other tools.

This is a Investigate Direction

Use the adx-direction-extra class to alert students to tasks that ask them to investigate, search and find information. Useful for research tasks like accessing databases, papers or articles.

This is a Practice Direction

Use the adx-direction-practice class to alert students to activities where they practice skills, tasks and processes.

This is a Reading Direction

Use the adx-direction-extra class to alert students to reading tasks. This could include reading texts, articles and papers.

This is a Reflect Direction

Use the adx-direction-extra class to alert students to reflection tasks.

This is a Write Direction

Use the adx-direction-extra class to alert students to writing tasks.

Cues - Learning

An example should use the adx-cue-example cue. This is a visual clue to an example and does not require a heading explaining that to a student, but should be evident in the text itself.

When providing a definition for a specific term or concept you should use the adx-cue-definition cue. This will provide a visual cue to students that something important is being defined in the content.
